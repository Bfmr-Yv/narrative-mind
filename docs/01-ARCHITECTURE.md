## 数据流（Phase 1）

```
用户输入文本片段
    │
    ▼
简化编排器接收 UserAction
    │
    ├─► 角色引擎：查询角色在该情境下的行为预测
    │       │
    │       └─► 语料锚定层：向量检索相似行为模式（top-k=3）
    │
    ├─► 世界引擎：校验设定规则（修为、空间、力量体系）
    │
    ▼
简化一致性守卫：交叉验证两引擎输出
    │
    ├─► 冲突？→ 返回冲突报告给用户（🟡提醒 / 🔴阻断）
    └─► 一致？→ 更新工作记忆 + 返回建议卡片
```

## 事件驱动调度（Phase 1）

| 用户操作 | 触发引擎 | 说明 |
|---------|---------|------|
| 写新段落 | 角色引擎 | 实时行为预测 |
| 修改设定 | 世界引擎 | 设定一致性检查 |
| 章节定稿 | 角色 + 世界（并行） | 全量检查 |
| 手动触发检查 | 角色 + 世界 + 守卫 | 完整校验流程 |
