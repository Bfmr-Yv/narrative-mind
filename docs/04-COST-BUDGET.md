## 降级策略（三层熔断）

### 第一层：50% 预算用尽（$10）

- 关闭文辞引擎、读者预期引擎（非核心引擎）
- 语料检索 top-k 从 3 降为 2
- 减少一致性守卫检查频率（从每次检查降为章节定稿时检查）

### 第二层：80% 预算用尽（$16）

- 切换到本地 Qwen2.5-7B 处理所有常规任务
- 仅保留角色引擎 + 世界引擎 + 一致性守卫在云端运行
- 语料检索 top-k 从 2 降为 1
- 禁用级联修正器的深度分析，仅保留关键词扫描

### 第三层：95% 预算用尽（$19）

- 停止所有云端 API 调用
- 切换到纯规则模式（Tier 0）
- 仅允许本地模型的基础功能
- 发送熔断警告通知

### 月度重置

- 每月 1 日 00:00 重置配额
- 上月剩余配额不累积
