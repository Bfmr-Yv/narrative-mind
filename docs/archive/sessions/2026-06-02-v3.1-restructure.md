1. **级联修正器保留 Phase 1 路径**：Phase 1 做关键词扫描+待办清单（$0 成本），Phase 2+ 升级到因果推理
2. **文辞引擎英文名统一为 Prose Engine**：与简报和契约保持一致
3. **Phase 1 守卫确认为 3 维**：角色行为 + 世界规则 + 空间一致
4. **Tier 1 成本为 $0**：本地 Qwen2.5-7B，零 API 成本，与 ADR-012 对齐
5. **示例使用通用奇幻位阶**：Phase 1 不做修仙专用，Prompt 示例为类型占位
