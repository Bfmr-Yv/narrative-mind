# 用户语料目录

此目录用于存放你自己的语料切片文件。

## 使用方式

将你的语料 JSON 文件放入此目录，格式需符合以下规范：

```json
[
  {
    "slice_id": "your_id_001",
    "source": "作品名",
    "chapter": "章节名",
    "text": "切片正文内容（建议 500 字左右）...",
    "metadata": {
      "characters": ["角色A", "角色B"],
      "scene_type": "narration",
      "emotion": "neutral"
    }
  }
]
```

## 版权提醒

⚠️ **请勿将受版权保护的作品语料提交到公开仓库。**

- 仅放入你有权使用的文本（公版作品、你自己创作的内容、或已获授权的文本）
- 本项目默认仅包含公版语料（如《红楼梦》节选）
- 如果你 fork 了本项目，请确保你的语料不侵犯他人著作权
